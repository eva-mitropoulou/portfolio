At a glance

Data: 3,955 public HTE records
Methods: component features, random forest, uncertainty diagnostics

Data

The workflow uses a public Buchwald-Hartwig HTE benchmark with ligand, additive, base, aryl-halide labels, and measured yield percentage.

Methods

Reaction cleaning, categorical component features, random forest baselines, grouped component splits, empirical uncertainty checks, active-learning simulation, and ranking diagnostics.
