Pharma-Facing Flagship Projects

Three headline case studies for cheminformatics, AI-assisted drug discovery, antibody and protein informatics, synthesis-aware ML, and validation-focused molecular modeling roles.

Antibody Sequence ML And Existing-Record Prioritization

Public-data antibody sequence workflow for retrospective benchmarking and prioritizing existing records.

Methods
CoV-AbDab, grouped validation, source holdout, k-mer TF-IDF, CDR annotation, antibody language-model benchmarks.
Key result
Selected whole-pair k-mer TF-IDF logistic regression as the defensible broad scorer: grouped ROC-AUC 0.780 and PR-AUC 0.823; source-robust ROC-AUC 0.610 and PR-AUC 0.636.
Project page Figure

EGFR CADD/QSAR Decision Workflow

Retrospective EGFR workflow with ChEMBL curation, Morgan RF baselines, scaffold validation, uncertainty checks, triage, and 5UG9 redocking.

Methods
RDKit, Morgan fingerprints, scaffold split, assay/document split, conformal intervals.
Key result
Morgan RF scaffold split RMSE 0.871/R2 0.550; 5UG9 redocking recovered the co-crystal ligand at 0.968 A RMSD.
Project page Repo

Reaction Yield Prediction And Synthesis-Aware ML

Retrospective public-data HTE benchmark with reaction cleaning, categorical component features, out-of-component validation, uncertainty diagnostics, active-learning simulation, and existing-record ranking.

Methods
Buchwald-Hartwig HTE data, categorical component features, additive-held-out validation, tree and linear baselines.
Key result
Selected random forest on additive-held-out validation: MAE 10.754, RMSE 14.237, R2 0.726, Spearman 0.860.
