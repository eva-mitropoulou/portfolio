Pharma-Facing Flagship Projects

Three headline case studies for cheminformatics, AI-assisted drug discovery, antibody/protein informatics, synthesis-aware ML, and validation-focused molecular modeling roles.

Antibody Sequence ML And Existing-Record Prioritization

Public-data antibody sequence workflow for retrospective benchmarking and prioritizing existing records.

Methods
CoV-AbDab, grouped/source-holdout validation, k-mer TF-IDF, CDR annotation, antibody LM benchmarks.
Key result
Selected whole-pair k-mer TF-IDF logistic regression as the defensible broad scorer: grouped ROC-AUC 0.780 / PR-AUC 0.823; source-robust ROC-AUC 0.610 / PR-AUC 0.636.
Limitation
Retrospective public labels with source/study effects; prioritization, not therapeutic design or prospective prediction.

EGFR QSAR / CADD Benchmark

Retrospective EGFR pIC50 baseline using ChEMBL curation, RDKit descriptors, and Morgan fingerprints.

Methods
RDKit descriptors, Morgan fingerprints, random split, Bemis-Murcko scaffold split.
Key result
Morgan random forest dropped from RMSE 0.712 / R2 0.719 to RMSE 0.871 / R2 0.550 under scaffold split.
Limitation
Baseline retrospective QSAR; not a production-grade predictor.
Project page Figure

Reaction Yield Prediction And Synthesis-Aware ML

Retrospective public-data HTE benchmark with reaction cleaning, component features, out-of-component validation, uncertainty calibration, active-learning simulation, and existing-record ranking.

Methods
Buchwald-Hartwig HTE data, categorical component features, grouped validation, tree and linear baselines.
Key result
Selected random forest on grouped validation: MAE 10.754, RMSE 14.237, R2 0.726, Spearman 0.860.
Limitation
Retrospective public-data benchmark only; existing-record ranking, not operational synthesis guidance.
