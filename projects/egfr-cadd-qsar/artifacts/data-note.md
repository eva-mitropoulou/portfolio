# Data Note

The full working project used public ChEMBL EGFR IC50 records and generated raw/processed descriptor tables. This portfolio copy intentionally excludes raw ChEMBL downloads and large intermediate processed datasets.

Included here:

- source scripts in `code/`
- selected model metrics in `results/`
- top-ranked candidate tables in `results/`
- final markdown reports in `reports/`
- representative figures in `figures/`

The workflow can be regenerated from the scripts using the dependencies in `requirements.txt`. The ChEMBL target used for the case study is EGFR target `CHEMBL203`.
